# Royal Star

A React-based card game practice application built with modern technologies.

## ⚠️ Important Notice

**This application is for amusement and entertainment purposes only.**  
No real money, prizes, or gambling is involved. This is a practice and learning platform.

## Technologies Used

This project is built with:

- **Vite** - Fast build tool and development server
- **TypeScript** - Type-safe JavaScript
- **React** - UI library
- **shadcn-ui** - Component library
- **Tailwind CSS** - Utility-first CSS framework
- **Supabase** - Backend and database

## Getting Started

### Prerequisites

- Node.js (v16 or higher)
- npm or yarn

### Installation

1. Navigate to the project directory:
```sh
cd betting-guardian-main
```

2. Install dependencies:
```sh
npm install
```

3. Set up environment variables:
   - Copy `.env` file and configure your Supabase credentials

4. Start the development server:
```sh
npm run dev
```

The application will be available at `http://localhost:8080`

## Available Scripts

- `npm run dev` - Start development server
- `npm run build` - Build for production
- `npm run build:dev` - Build for development
- `npm run preview` - Preview production build
- `npm run lint` - Run ESLint
- `npm test` - Run tests
- `npm run test:watch` - Run tests in watch mode
